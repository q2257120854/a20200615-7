#### 安装教程
1、将源码解压到服务器空间
2、域名应该指向到public目录，因为应用入口文件位于public/index.php。
比如我的DSCMS项目在  D:\www\DSCMS  域名应该指向到 D:\www\DSCMS\public
3、进行安装  http://域名/install/install.php
4、后台地址：http://域名/index.php/admin
5、前台地址：http://域名/index.php/home
